#PROJECTS


SoundScapes was an experiment in interactive design skills, artistic explorations, and User Experience (UX) research techniques. The project was inspired by Buddhism, biocentrism, and indie rock.

Version 1, completed in Winter 2016, was built to visualize voice and ambient noise in Google Chrome, using HTML, CSS, JavaScript, Web Audio API, and Three.js.

Version 2, which is currently in progress, had the same goals but is being developed for Google Cardboard through the Unity game engine.
